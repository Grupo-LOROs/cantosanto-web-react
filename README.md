# Cantosanto Web (React + Vite + Tailwind)

Implementación en React basada en la captura que compartiste (estructura, tipografía y layout responsive).
Los colores/tipografías están aproximados; si me pasas los valores exactos del Figma (o el style guide),
los ajusto para que sea 1:1.

## Requisitos
- Node.js 18+ (recomendado)

## Instalación
```bash
npm install
```

## Desarrollo
```bash
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
